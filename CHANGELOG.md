# Changelog

All notable changes to this project will be documented in this file.

## [v1.0.0] - 2026-03-04
### Highlights
- Added quest depth expansion with 33 quests, branching prerequisites, and multi-objective progression.
- Added endgame challenge loop with repeatable scaling encounters and streak/tier rewards.
- Added rotating endgame challenge modifiers (risk/reward variants and cycle bonus rewards).
- Added quarterly seasonal event system with encounter/quest reward modifiers.
- Added seasonal repeatable quest set gated by active seasonal event windows.
- Added live balance patch note generator (`npm run balance:notes`) with dated markdown snapshots.
- Added first-clear playtime balance validator targeting a 12-hour average campaign.
- Added one-command launcher (`npm run play`) for install/build/start flow.
